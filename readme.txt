# pbcommand plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

pbcommand plugin (v 1.0.5) for B3

Installation:

1. Place the pbcommand.py in your ../b3/extplugins and the 
pbcommand.xml in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

   <plugin name="pbcommand" config="@b3/extplugins/conf/pbcommand.xml"/>
